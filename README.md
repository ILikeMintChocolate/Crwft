# crwft / Client

크래프트의 클라이언트 파일입니다.

## 실행방법
설치
```
git clone https://github.com/ILikeMintChocolate/Crwft.git
cd Client
npm install 
npm install fabric
npm install firebase
npm install svelte-spa-router
```
실행
```
npm run dev
```
