# crwft / Compile Server

크래프트의 파일 컴파일 및 저장 서버입니다.

## 실행방법
설치
```
git clone https://github.com/ILikeMintChocolate/Crwft.git
```
빌드
```
cd Compile\ Server
chmod +x gradlew
./gradlew build
```
실행
```
cd build
cd libs
java -jar crwft-file-server-0.0.1-SNAPSHOT.jar
```
